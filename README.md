# README 작성을 위한 Markdown 활용법
### Header(제목)
* #의 갯수에 따라 h1~h6까지 지원.
* 사용법 : #(띄어쓰기)글
```
# This is an H1
## This is an H2
### This is an H3
#### This is an H4
##### This is an H5
###### This is an H6
```
# This is a H1
## This is a H2
### This is a H3
#### This is a H4
##### This is a H5
###### This is a H6
####### This is a H7(지원하지 않음)

***

### Emphasis(강조)
* ```[* | _]글[* | _]``` : 기울기
* ```_
*This text will be italic*
_This will also be italic_

**This text will be bold**
__This will also be bold__

_You **can** combine them_
