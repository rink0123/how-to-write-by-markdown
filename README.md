# README 작성을 위한 Markdown 활용법
### Header(제목)
* #의 갯수에 따라 h1~h6까지 지원.
* 사용법 : ```#(띄어쓰기)글```
```
# h1 tag
## h2 tag
### h3 tag
#### h4 tag
##### h5 tag
###### h6 tag
```
# h1 tag
## h2 tag
### h3 tag
#### h4 tag
##### h5 tag
###### h6 tag

***

### Emphasis(강조)
* ```*``` 또는 ```_```의 갯수가 1개냐 2개냐에 따라 글자 기울기 또는 글자 굵기로 나뉨.
* 사용법 : ```[* | _]글[* | _]``, ```[** | __]글[** | __]```
*i tag*
_i tag_
**b tag**
__b tag__

_You **can** combine them_
